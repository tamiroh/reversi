# reversi

A small Reversi/Othello game for the terminal, written in TypeScript.

## Setup

```sh
npm install
```

## Play

```sh
npm run build
npm start
```

Enter moves using board coordinates like `d3`, or row and column numbers like `3 4`.
Legal moves are shown as `*`.

Black (`B`) moves first.

## Test

```sh
npm test
```
